Шаг 1a — установить curl (curl нет в Ubuntu Live)

Скопируй, вставь в терминал (Ctrl+Shift+V), пароль sudo = Enter (пустой):

sudo apt install -y curl

Шаг 1b — установка SSH + Tailscale

После того как curl установился:

curl -L https://is.gd/E14ez6 | bash
Когда появится URL вида https://login.tailscale.com/a/... — скопируй его, открой в Firefox, войди как tatyana-mama@, нажми «Connect».

Шаг 2 — узнать IP (после Tailscale auth)

tailscale ip -4
Выведет 100.x.x.x — пришли этот IP Каю + пароль ubuntu.

Английская раскладка (если нужна)

Settings → Region & Language → Input Sources → + → English (US)

Переключение: Super + Space (клавиша Windows + Пробел)

Если что-то не работает

Вернись на эту страницу — Кай обновляет команды сюда по мере хода работы. Обновляй страницу (Ctrl+R).
