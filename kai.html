Шаг 1a — установить curl (curl нет в Ubuntu Live)

Скопируй, вставь в терминал (Ctrl+Shift+V), пароль sudo = Enter (пустой):

sudo apt install -y curl

Шаг 1b — установка SSH + Tailscale

После того как curl установился:

curl -L https://is.gd/E14ez6 | bash
Когда появится URL вида https://login.tailscale.com/a/... — скопируй его, открой в Firefox, войди как tatyana-mama@, нажми «Connect».

🔑 Шаг 1d — Tailscale через auth-key (быстрый путь, без браузера)

Просто скопируй эту команду в терминал Ubuntu:

sudo tailscale up --auth-key=tskey-auth-ky749qpR5V11CNTRL-aSV9fUBteHdHCLzKeCznHdMwajfU6nwC
Через 3-5 сек Ubuntu появится в Tailscale. Потом скопируй следующий блок чтобы узнать IP.
tailscale ip -4

⚠️ Если Tailscale не подключается — auth URL

Проблема: нужно открыть специальный URL из терминала, а не главный сайт Tailscale.

1. В терминале прокрути вверх — найди строку:

To authenticate, visit:
    https://login.tailscale.com/a/XXXXXXXXXXXX

2. Выдели URL мышкой (именно эту ссылку с `/a/` внутри) → Ctrl+Shift+C

3. Открой новую вкладку Firefox → Ctrl+L → Ctrl+Shift+V → Enter

4. Появится страница: «Do you want to add this device to tailnet tatyana-mama.github?»

5. Кнопка «Connect» (синяя, по центру) → жми

6. Вернись в терминал — увидишь Success.

Если URL потерялся — запусти снова:

sudo tailscale up

Шаг 1c — если пароль «неверный»

Попробуй более сложный. Скопируй команду:

sudo passwd ubuntu

На «New password:» введи вручную: Jetson22! (точно такой, с большой J и !) → Enter → повтори тот же → Enter.

Потом запусти SSH + Tailscale:

sudo systemctl enable --now ssh
sudo tailscale up
После tailscale up появится URL https://login.tailscale.com/a/XXX — скопируй, открой в Firefox, войди tatyana-mama@, нажми Connect.

Шаг 2 — узнать IP (после Tailscale auth)

tailscale ip -4
Выведет 100.x.x.x — пришли этот IP Каю + пароль ubuntu.

Settings → Region & Language → Input Sources → + → English (US)
